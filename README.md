# ProductCarousel

A simple and responsive product carousel built with frontend technologies.

## Description
This project showcases a product carousel UI component that allows users to browse through products interactively.

## Features
- Responsive design
- Smooth sliding animation
- Easy to customize

## Installation
1. Clone the repository:
2. Open the project folder in your favorite code editor.
3. Open `index.html` in a web browser to see the carousel in action.

## Usage
Modify the product data in the JavaScript file to add or update products shown in the carousel.
